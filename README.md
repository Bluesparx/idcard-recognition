## Setup: 
1. Open two empty terminals and cd to backend and frontend
3. On backend, run these commands. It will run on port 5000.
```
python -m venv venv
venv/Scripts/Activate
pip install -r requirements.txt
python app.py
```
4. On frontend, run these commands:
```
npm install
npm run dev
```
